singly_linked_lists
This directory contains functions to manipulate singly linked lists in C.

Files
0-print_list.c: Function that prints all the elements of a list_t list.​

1-list_len.c: Function that counts the number of elements in a linked list.​

2-add_node.c: Function that adds a new node at the beginning of a linked list.​

3-add_node_end.c: Function that adds a new node at the end of a linked list.​

4-free_list.c: Function that frees a linked list.​

lists.h: Header file containing the list structure definition and the function prototypes.​

Usage
Include lists.h in your programs to use the singly linked list manipulation functions, and compile the files with gcc.​
